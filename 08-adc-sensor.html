ADC 与传感器采集

真实世界的温度、光照、电压是模拟量，MCU 通过 ADC（模数转换器）将其变为数字值。

ADC 关键参数

  • 分辨率：常见 12 位，即 0～4095。
  • 参考电压 Vref：如 3.3V，则 LSB ≈ 3.3/4096 V。
  • 采样时间：高阻抗信号源需更长采样时间。

电压换算

uint16_t raw = HAL_ADC_GetValue(&hadc1);
float voltage = raw * 3.3f / 4095.0f;

数字传感器 vs 模拟传感器

  • 数字：DHT11、BME280 等通过 I2C/UART 输出已处理数据，开发快。
  • 模拟：LM35 等输出电压，需 ADC + 标定，成本低。

简单滤波

单次采样易受噪声影响，常用滑动平均或中值滤波：

uint16_t moving_average(uint16_t new_sample) {
    static uint16_t buf[8];
    static uint8_t idx = 0;
    buf[idx++ & 7] = new_sample;
    uint32_t sum = 0;
    for (int i = 0; i < 8; i++) sum += buf[i];
    return (uint16_t)(sum / 8);
}
