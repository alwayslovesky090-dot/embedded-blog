GPIO 与基本外设

GPIO（General Purpose Input/Output）是最常用的外设，用于读按键、控 LED、片选传感器等。

输出模式

  • 推挽输出（PP）：可主动输出高/低，驱动 LED、继电器等。
  • 开漏输出（OD）：只能拉低或浮空，常配合上拉用于 I2C 等总线。

输入模式

  • 浮空输入：电平由外部电路决定，需外部上下拉。
  • 上拉 / 下拉输入：无外部信号时有默认电平，适合按键检测。

按键检测与消抖

机械按键按下时触点会抖动，需软件消抖：

uint8_t read_key_stable(void) {
    if (HAL_GPIO_ReadPin(KEY_GPIO, KEY_PIN) == GPIO_PIN_RESET) {
        HAL_Delay(20);  // 消抖延时
        if (HAL_GPIO_ReadPin(KEY_GPIO, KEY_PIN) == GPIO_PIN_RESET)
            return 1;
    }
    return 0;
}

简单驱动分层

良好习惯是将硬件细节封装成模块 API：

void led_on(void);
void led_off(void);
void led_toggle(void);
uint8_t key_pressed(void);

应用层只调用 led_on()，换板子时只需改底层实现。
