第一个程序：点亮 LED

点亮 LED 是嵌入式领域的「Hello World」。它验证了：编译 → 下载 → 硬件控制 整条链路。

硬件连接

LED 长脚接 GPIO（通过限流电阻，通常 220Ω～1kΩ），短脚接 GND。若 LED 接 VCC，则 GPIO 输出低电平时点亮（灌电流方式）。

步骤概览

  1. 使能 GPIO 端口时钟（STM32 外设默认时钟关闭以省电）。
  2. 配置引脚为推挽输出模式。
  3. 设置输出高或低电平，使 LED 亮或灭。

寄存器方式（概念示例）

// 1. 开启 GPIOA 时钟
RCC_APB2ENR |= (1 << 2);

// 2. 配置 PA5 为推挽输出 50MHz（具体位域见手册 CRL 寄存器）
GPIOA_CRL = (GPIOA_CRL & ~(0xF << 20)) | (0x3 << 20);

// 3. 置位 PA5，LED 亮
GPIOA_BSRR = (1 << 5);

HAL 库方式（STM32）

int main(void) {
    HAL_Init();
    SystemClock_Config();

    __HAL_RCC_GPIOA_CLK_ENABLE();
    GPIO_InitTypeDef gpio = {0};
    gpio.Pin = GPIO_PIN_5;
    gpio.Mode = GPIO_MODE_OUTPUT_PP;
    gpio.Speed = GPIO_SPEED_FREQ_LOW;
    HAL_GPIO_Init(GPIOA, &gpio);

    while (1) {
        HAL_GPIO_WritePin(GPIOA, GPIO_PIN_5, GPIO_PIN_SET);
        HAL_Delay(500);
        HAL_GPIO_WritePin(GPIOA, GPIO_PIN_5, GPIO_PIN_RESET);
        HAL_Delay(500);
    }
}

常见问题

  • LED 不亮：检查引脚号、电阻、共阳/共阴接法。
  • 程序跑飞：检查时钟配置、是否启用 HardFault 调试。
